# 🚀 Quick Start

📄 Clone or fork `payc-marketplace`:

```sh
git clone https://github.com/Phunky-Ape-Yacht-Club/phunky-mp.git
```

💿 Install all dependencies:

```sh
cd payc-marketplace
yarn install
```


🚴‍♂️ Run your App:

```sh
yarn start
```
